# 📈 API REST Histórico IPCA

## 📖 Sobre o Projeto

API REST desenvolvida para consulta do histórico do IPCA, utilizando Node.js e Express, com foco em integração de rotas, manipulação de dados e desenvolvimento Back-end.

---

## ⚙️ Funcionalidades

- 📊 Consulta do histórico do IPCA
- 🔗 Rotas REST
- ⚡ Retorno de dados em JSON
- 🛠️ Estrutura organizada de API
- 📡 Testes de requisições com Postman

---

## 🛠️ Tecnologias Utilizadas

- Node.js
- Express
- JavaScript
- Postman

---

## 🎯 Objetivo

Projeto desenvolvido para praticar desenvolvimento de APIs REST, estruturação de rotas e manipulação de dados no Back-end.
